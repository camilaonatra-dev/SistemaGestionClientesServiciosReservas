# Sistema Integral de Gestión de Clientes, Servicios y Reservas

Este proyecto implementa un sistema orientado a objetos para gestionar clientes, servicios y reservas para la empresa Software FJ. Ofrece servicios como reservas de salas, alquiler de equipos y asesorías especializadas.

## Características

- **Programación Orientada a Objetos**: Utiliza clases abstractas, herencia, polimorfismo y encapsulación.
- **Manejo de Excepciones**: Incluye excepciones personalizadas y logging robusto.
- **Sin Base de Datos**: Toda la información se maneja en memoria con listas y archivos para logs.
- **Validaciones**: Validaciones estrictas en datos y operaciones.

## Estructura del Proyecto

- `src/clases.py`: Definición de clases (Cliente, Servicio, Reserva, etc.)
- `src/main.py`: Simulación de operaciones
- `logs.txt`: Archivo de logs para errores y eventos

## Cómo Ejecutar

1. Navegar a `src/`
2. Ejecutar `python main.py`
